- **Backend API**: http://localhost:5000/api

### 4. Test Everything

## 📊 API Endpoints

### Market Data
```
GET /api/market/summary          # Market overview
GET /api/market/top-gainers      # Top gaining stocks
GET /api/market/top-losers       # Top losing stocks
```

### Stock Information
```
GET /api/stocks                  # List all stocks
GET /api/stocks/{symbol}/data    # Historical data
GET /api/stocks/{symbol}/predict # AI prediction
GET /api/stocks/{symbol}/report  # Full analysis
```

### Portfolio
```
POST /api/portfolio/simulate     # Portfolio simulation
POST /api/models/train           # Train ML models
```
